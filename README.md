# Yelppy

A Yelp clone web app

## Our stack
1. ReactJS
2. Firebase
3. Google Map API
4. Yelp API

## Run instruction

1. Clone or download our application
2. Install npm
3. Install node
4. Install firebase
	npm install firebase --save-dev
5. Install react-router
	npm install react-router --save-dev
6. To install dependencies
	npm install
7. To run
	npm start
